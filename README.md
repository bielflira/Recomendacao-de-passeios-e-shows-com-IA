# Recomendacao-de-passeios-e-shows-com-IA
Descubra Rolês e Shows com IA.  O meu app é um aplicativo inteligente que recomenda passeios, eventos e shows personalizados com base nos gostos do usuário. Utilizando Inteligência Artificial e integração com APIs de eventos reais, focado em experiências do mundo real.
